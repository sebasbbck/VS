Este proyecto fue una petición por parte de una profesora. Básicamente sirve para gestionar las incidencias dentro de las aulas del instituto.
He utilizdo XAMPP.
Se hace uso de Oauth para poder iniciar sesión con una cuenta de google y como esta página es para nuestro centro, solo se inicia sesión con cuentas que acaben en "@educa.madrid.org". Incluye PhpMailer el cual envía
correos mediante el servidor SMTP de educamadrid. Los correos se envían a los usuarios, en este caso son profesores, los cuales estén asignados a las aulas donde se cree una incidencia.
